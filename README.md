# ElevenLabs Hackathon 2025

## Description

The application helps you improve your diagnosis skills by talking (or chatting) with patients based on case studies.

## Technologies

* ElevenLabs API
* OpenAI API
* FastAPI
* n8n
* Next.js
* Tailwind CSS
